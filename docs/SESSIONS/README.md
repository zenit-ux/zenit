# SESSIONS — Claude Code Session Logs

A dated log of every Claude Code work session on Zenit.

---

## Purpose

Track what was built, what was decided, and what's next — so context is never lost between sessions.

---

## How to Use

1. At the end of each session, duplicate `SESSION-TEMPLATE.md`
2. Rename it: `YYYY-MM-DD-short-task-name.md`
3. Fill in all sections (takes ~3 minutes)
4. Commit alongside the work

---

## Archive Strategy

When the active log gets long (>20 sessions), move older files to `ARCHIVE/`:

```
docs/SESSIONS/
├── ARCHIVE/
│   └── [sessions older than ~3 months]
├── README.md
├── SESSION-TEMPLATE.md
└── [recent sessions]
```
